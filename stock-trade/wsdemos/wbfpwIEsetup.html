How to setup your IE to use FPW:
 First of all, FPW has been developed for Microsoft IE English version. We are not sure if all FPW functions are working for other browsers. If you could not open some FPW pages every time, or have problem to login or the showing page is messing up, for the most of the times, it is because your Internet Explorer did not setup properly for using FPW. Please check the following settings:

Under IE-View-Text Size, to choose Medium.
Under IE Internet Properties/Options - Security - Internet - (Clicking Custom Level to) Miscellaneous:
     1, To choose Enable for "Allow websites to open windows without address or Status bars"
     2, To choose Disable for "Use Pop-up Blocker" or add www.WSQSI.com to your allowing list
     3, To choose Enable for "Allow status bar updates via script”, in order to view on-line help message.

To do the same things to your Anti-Virus software.
