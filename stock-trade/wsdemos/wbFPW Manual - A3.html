SWFDT (Swing STAT)

 function [y y1]=swfd(stock,level,picture)
                stock : data to be analyzed; format: [h l c].
                level : swing level, usually < 20

    picture:  0 - pivots mark at where happened,
                               others: mark at where confirmed, this will bring delay, but it is the real case.
                y(:,1): Confirmed pivot before this bar -- 0: outside points, -1: low, 2: high
                y(:,2): Confirmed previous pivot point is how many bars away from this bar
                y(:,3): Confirmed 2nd previous pivot point is how many bars away from this bar
                y(:,4): Confirmed 3rd previous pivot point is how many bars away from this bar
                y1 : == swfd(stock,level), pivot location -- 1: outside points, -1: low, 2: high, 0: nothing

 

SWFDT1 (Swing STAT 1)

function [y yt stockt yc] = swfdt1(stock,level)
                stock : data to be analyzed; format: [mm dd yy hh mm o h l c v] or [mm dd yy o h l c v].
                It should be much longer than level, suggested length is 50*level.
                level : swing level, usually < 20
                y : [y1 y2 y3 y4 y5 y6 y7 y8 y9 y10 y11 y12] for the latest day's bars
                    y1 : 1 - High is the last Pivot
                           0 - outside bar is the last pivot
                          -1 - Low is the last Pivot
                    y2 : Latest High Pivot-price
                    y3 : Latest High Pivot-place
                    y4 : Latest Low Pivot-price
                    y5 : Latest Low Pivot-place
                    y6 : Second Latest High Pivot-price
                    y7 : Secong Latest High Pivot-place
                    y8 : Second Latest Low Pivot-price
                    y9 : Secong Latest Low Pivot-place
                    y10: lines Trend or direction, 1: up (y2>=y6)&(y4>=y8), -1: down (y2<y6)&(y4<y8), 0: other cases.
                    y11: Trend's 'resistant' (upper line) price level for this bar index/time
                    y12: Trend's 'support' (lower line) price level for this bar index/time
                yt : last day's y only. many rows x 12 for intraday, or 1x12 row for daily data
                         Its pivot place index will also change to current data serial, so minus index is possible and
                         it means that pivot places are happened before 'today'. 1 means today's first bar.
                stockt: last day's data [mm dd yy hh mm o h l c v] (many rows) or [mm dd yy o h l c v] (1 row only)
                yc : 1 row for last bar only, no matter what data type, day or intraday
                if stock is daily bar x8, y is just an one 1x12 vector.
                if stock is intraday bars x10, size(y) will be size(c)x12 matrix, each
                    row contains its corresponding y for that bar time for the last day. Zeros for other days.
